Analizador de tono y speech-to-text

Práctica final de Arquitecturas de Servicios en Red de Paula de Vega Botas y Gimena Segrelles Munárriz

Qué microservicios ofrecemos

Analizamos el tono de un texto; puede ser una entrevista, una frase que has oído por la calle o una conversación de película. Puedes dictarnos el texto o puedes teclearlo.

Analizar el texto

Nuestro proyecto

Analizador de texto, ya sea escrito o dictado (insertado desde un archivo WAV). El texto puede ser insertado en varios idiomas, como español o inglés. Por último, se podrá guardar en la base de datos la información procesada (el texto analizado y los resultados).
